# 🌾 Vivasayi Mitra — விவசாயி மித்ரா

AI-powered crop disease detection for Tamil Nadu farmers.

## Features
- 🌿 Detects 15 plant diseases
- 🗣️ Tamil & English support
- 💊 Remedy suggestions
- 90%+ accuracy (EfficientNetV2-S)

## Supported Crops
🌶️ Pepper | 🥔 Potato | 🍅 Tomato

## Model
- Architecture: EfficientNetV2-S
- Dataset: PlantVillage (20,639 images)
- Training: PyTorch + Kaggle P100 GPU
